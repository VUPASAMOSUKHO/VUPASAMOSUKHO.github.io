中日两国合作学习理论的比较研究

郑友训 / 刘效川

关键词：合作学习; 学习共同体; 佐藤学; 论文; 王坦; 比较研究; 中日两国

中日两国在合作学习理论的研究与应用方面各有特点，比较其中的异同，为进一步探究合作学习提供借鉴。王坦与佐藤学分别是中国和日本合作学习理论研究的代表人物，从学习观、教学观、师生观、教学实践等四个维度，梳理二者的合作学习理论，比较其异同，并从中总结出五点启示：激活教学中的默会知识，重视教学中的互动，关注学生的多元发展，对教师提出新的要求，发展完善理论体系。

合作学习在当前的教学改革中被广泛应用，然而，诸多教学实践表明，我国的合作学习仍然没有脱离传统教学的桎梏，阻碍了教学的进一步变革与发展。梳理合作学习的核心要素，对中日两国合作学习理论研究的比较、分析，能为合作学习研究提供启示。在合作学习研究领域，王坦与佐藤学分别是中国和日本具有代表性的研究学者，对二者的合作学习理论进行比较、分析，可以直观地厘清中日两国的合作学习研究异同。

王坦和佐藤学的合作学习研究

合作学习发端于美国，20世纪70年代，美国教育界针对传统班级授课制的一些局限，开展了合作学习的研究、实验，在全球教育改革中备受瞩目。王坦曾在北美研修合作学习；佐藤学的博士论文研究的是美国教育问题。因此，王坦与佐藤学的合作学习研究不可避免地受到美国合作学习理论的影响，而且在很大程度上，他们要借鉴美国教育学者对合作学习理论与实践的研究，批判传统课堂教学的弊端，推进课堂教学改革。

王坦与合作学习王坦是我国合作学习理论的知名学者，也是我国最早研究合作学习的学者之一。研究专著有《合作学习导论》（1994）、《合作学习——原理与策略》（2001）、《合作教学导论》（2007）等，最具代表性的期刊论文有《合作学习述评》《论合作学习的基本理念》《合作学习简论》《合作学习的理论基础简析》《论合作学习的教学论贡献》等。王坦全面论述了合作学习的概念界定、理论基础、基本理念和教学策略，有效指导了全国数十个合作教学研究基地工作的顺利进行，这为我国合作学习的理论研究和教学实践打下了坚实的基础，发挥了重要的作用。

佐藤学与合作学习佐藤学是日本著名的教育学者，是日本“学习共同体”的创建者，以合作学习为教学方式展开“静悄悄的课堂革命”，进行了大量的学校改革实践，并获得了巨大成功。研究论述主要集中在《静悄悄的革命——创造活动、合作、反思的综合学习课程》（2003）、《学校的挑战——创建学习共同体》（2010）、《教师的挑战——宁静的课堂革命》（2012）等专著中，最具代表性的期刊论文有《论“教学的创造”》《教学范式与课程文化》等。佐藤学构建了基于“学习三位一体论”的新型课堂教学范式，通过“润泽”的课堂文化，进行学校改革，最终实现教育整体的变革与完善。

王坦与佐藤学合作学习理论比较

在合作学习本土化研究过程中，王坦和佐藤学提出了不同的见解，分别构建起具有本国特点的合作学习理论。

关于学习观

王坦注重激发学生的学习动机，秉持学生要产生内在“愿意学”的需要，就能够实现“学得好”的效果。在合作学习小组中，积极互赖能够激发学生的学习动机。建立积极互赖的途径有两个：一是在合作学习小组成员之间进行学习任务分工和不同角色的扮演，学生各自负责不同的学习内容和小组任务；二是小组的奖励结构，把每个学生的成绩用改进的“标准参照”算出的分数总和作为小组成绩，参与合作学习小组的评比。在竞争、合作、个体三种学习情境中，浓厚的竞争色彩扭曲了学生之间应有的合作意识与合作能力，然而，学生需要学会合作。合作学习是优化三种学习情境的组合：竞争、合作和个体，学生通过合作实现更好的学习。

佐藤学提出了“三位一体的学习观”，认为学习是同教材、他人和自己的相遇与对话。通过合作，学生之间产生多元思想碰撞，从而雕琢自己的思想，形成自己的学习。佐藤学批判了传统的同步教学。他认为在同步教学中只有三分之一学生在保持“学习状态”，因此，他提倡把教学内容水准设定高一点，但同时在教学中必须照顾到低水准学生，建立“以学习为中心的教学”，填平所有学生水准之间差异需要教师与学生的合作学习，“上、中、下”三层学生都能实现学习的“冲刺与挑战”，要保证学生挑战高层次的学习机会，实现每个学生的“挑战与冲刺”的学习权利，这是合作学习的本质。王坦和佐藤学都注重调动学生学习的积极性，激发学生的学习动机，二者在策略选择上有所不同：王坦选择从外向内的方法，通过小组成员之间的积极互赖唤起学生个体的学习动机，为了实现小组的目标，学生需要承担自己的个体责任；佐藤学选择从内向外激发学生的学习动机，通过追求适当高水准的“冲刺与挑战性学习”，激发所有学生的学习动机，保持每一位学生的学习状态。二者均是通过激发学生的内在动机，形成主体意识，为学生的学习提供了充足的动力源泉。

关于教学观

王坦对合作学习的定义为：合作学习是以学习小组为基本形式，采用动态交互的教与学，评价小组的团体成绩，共同达成教学目标的活动。这个被广泛引用的合作学习的定义，既表达了“合作”的学生学习状态，也表达了合作学习是一种以学生合作学习为基础的教学方法，动态因素为合作学习增加了复杂性，师生之间以教学内容为媒介进行全员互动，教学的目标性要求教师在互动过程中负有目标导向的责任，并对合作学习小组进行评价。在“合作教学研究与实验”中，王坦尝试提出合作学习的一般模式：设计—呈现—活动—测验—奖励—反馈。教师对教学整体设计，结合集体授课与合作学习，把合作学习小组分为组间同质、组内异质，学生之间的合作能唤起并保持学生的学习，教师主要是以评价、反馈的形式参与其中。

佐藤学认为传统课堂教学为“座学式学习”课堂教学，应该放进博物馆，因为这是一种教学失范。他认为，“座学式学习”是模仿范式、传递范式，这两种范式以行为主义学习理论为基础。模仿范式表示学生对知识的学习是一种模仿，是同步教学的结果，要求“回答正确”甚于“理解意义”，学生学习的知识属于奥苏贝尔的“无意义学习”；传递范式中，教师把知识作为既定的“物”传递给被当作容器来对待的学生，对学生学习的要求是记忆。佐藤学强烈批判这种落后的教学，提倡合作学习。与模仿范式相对应的是变革范式，与传递范式相对应的是对话范式。学习的个性化和创造性是变革范式的核心，关注学习者与客体、他人、自己的相遇与对话，是真正的学习；对话范式强调师生间的“发散型提问”，一问多答，是师生共同探究型教学。佐藤学在日本丰玉南小学研讨会上纠正一位教师的合作学习模式，应该是：交响的学习、合作的学习、澄明的学习。其中，澄明的学习是指学生在知识内容和个人发展方面都得到了提高。合作学习是师生之间学习的智慧碰撞，学生个体无论在学习共同体还是自主学习的情境中，都能实现“冲刺与挑战”的学习。

王坦和佐藤学都批判传统教学的落后性，试图构建以合作学习为基础的新型课堂范式。王坦的“合作学习模式”把小组之间的合作学习作为课堂教学中的一个环节，创新了教学组织形式，只不过，教师仍然控制着课堂教学的秩序与进度，师生之间的合作、互动局限于寻找“标准答案”，没能脱离传统教学的桎梏。佐藤学提倡以倾听与对话重新构建课堂教学范式，对传统教学模式有巨大的冲击，有利于革新传统课堂教学模式的诸多弊端，让教学更接近学与教的本质过程。

关于师生观

王坦认为教师在教学中主要是处理学生认识过程的矛盾，也就是说，师生之间的关系在教师帮助学生学习的过程中建立起来。教师当“导演”，学生当“演员”，教师与学生导、演分工，不仅能保证教师导学的质量，也能保证学生的主体性。教师成为学生学习的促进者。合作学习的模式中，合作设计、集体呈现、团体奖励和反馈补救主要由教师主导完成，教师的环节设计是合作学习有效开展的前提，在集体讲授的基础上组织合作环节，教师对各合作小组多元的学习结果进行评价、奖励。合作学习是全员合作，要激活师生本身的资源价值，而不是仅限于师生间的单向互动。学生在合作学习小组中，分担不同任务参与小组合作，向班级展示小组合作学习结果的时候，学生个人代表整个小组与教师或者别组学生对话、交流。师生关系从“权威—服从”逐渐变成“指导—参与”的关系。[佐藤学认为学生在课堂上要做好：活动（作业）、合作学习、分享表达。活动(作业)是指学生自主学习，学生在学习上能够实现自主学习、合作讨论并与其他人分享表达，这打破了传统的“座学式学习”，实现了活动性、合作性、反思性的学习。

教师在教学中要做好：倾听、串联、反刍。佐藤学认为每一位学生的发言都是精彩的，教师仔细倾听是对学生之间差异性的尊重，也是对学生学习能力的尊重，“倾听”是教学中教师活动的核心。知识经过学生内化处理后外化为语言表达出来，具有其逻辑关系，学生的发言反映了他的内在思考，教师对学生学习的促进，就是通过倾听学生对知识处理的特殊逻辑关系，把握他的学习，从而指导该学生进一步的学习。在倾听过程中，把学生的表达与教材内容、其他学生的表达“串联”起来是教师的核心工作，这也是深化教学内容、实现“冲刺与挑战”学习的保障。反刍是将学生的表达串联起来与教材内容进行融合，推进教学进程。这些对教师的要求很高，为了提高教师的应对能力，佐藤学提倡在教师之间建立“同僚性”关系，敞开各自课堂的大门，相互听、评、学习。 王坦和佐藤学都强调教师与学生之间的互动，充分发挥学生的主体性，保护学生的差异，发展学生的个性和创造力。佐藤学提倡教师以倾听、串联与反刍“应对”学生活动（作业）、合作学习、分享表达，甚至喊出了“一切差异万岁”的口号，尊重学生个性，激发学生的创造性。

王坦和佐藤学都提倡教师之间相互合作。王坦从合作学习的互动因素论证师师合作的重要性，师师互动既能减轻教师个人教学负担，又能相互启发、思维碰撞，提高自身能力。佐藤学提倡教师之间建立“同僚性”合作关系，构筑“同僚性”的校本研究，教师由“教育专家”转型为“学习专家”。教师本身就是宝贵的教学资源，教师之间的合作，能够从集体智慧中受益，也能建立学习型的教师组织，提高教师的能力。

关于教学实践

王坦的合作学习研究立足于山东省中小学合作学习实验，指导了几十个合作教学重点研究基地的研究，参与总结了像杜郎口中学这样的合作教学典型。杜郎口中学的“三三六”模式一度风靡全国。佐藤学在日本协助近2000所中小学的改革，其中最具代表性的是滨之乡小学和岳阳初中。“滨之乡模式”的精髓是在师生之间构筑起“相互倾听”的“合作学习关系”。岳阳初中的经验是构建以“学”为中心的生活，在课堂上引入三个要素：活动（作业）、合作学习、表达分享。这两所学校的合作学习改革作为示范被日本数千所中小学校学习、借鉴。以王坦和佐藤学为代表的中日研究者对合作学习理论研究的差异，导致了中日两国合作学习在课堂教学中的表现大相径庭。我国的教学实践中，合作学习取得了一定成效，但是，合作学习应有的诸多优越性仍没能充分地释放出来，合作学习的理论与实践发展到了瓶颈阶段，“合作学习热”逐渐式微。日本的中小学正波澜壮阔地掀起以合作学习为基础的“宁静的课堂革命”，得到师生的广泛支持，取得了巨大成功。

几点启示

通过对王坦和佐藤学合作学习理论的比较，可以看出二者的研究既有共同之处，也有相异之处。反思这些异同点，使我们对合作学习形成更本质的认识，为我们进一步探究合作学习提供了宝贵的启示。

  1. 激活教学中的默会知识课堂中显性知识易于师生之间的教学，而默会知识则难以直接言表，在教师所教和学生所学之间横亘着一条难以穿越的沟壑。合作学习立足于激发每位学生的学习动机，唤起学生内在的学习需求，以各自独特的知识处理逻辑建构自己的知识学习。合作学习通过师生、生生的对话、交流、讨论等，促进学生领悟他人对知识不同的处理方式，引发学生反思自己的知识处理逻辑。合作学习为学生体悟默会知识架起了桥梁，激活了教学中难以触及的默会知识，有力地提高了学生的元认知能力。

  2. 重视教学中的交流互动合作学习既是一种学习方式，也是一种教学方式，双重属性使得合作学习具有教与学双方面的本质特征。在教学实践中，师生、生生之间通过“言为心声”的交流互动，师生从“心”开始教和学。学生之间展开以教学内容为导向的合作学习，集独立思考与合作交流于一体，学生的思维活动、知识理解相互碰撞、交融，促进了学生的课堂学习。师生之间的对话，让教师比以往有更多机会聆听学生的表达，把握学生对知识理解的内在逻辑。在课堂教学中，教师与学生在互动中“奏响交响乐”，激发学生自己建构知识的思维火花，突出合作学习的优势。

  3. 关注学生的多元发展合作学习重视发现学生内在的特征，以学生自身为出发点引导学生的多元发展。一是发展学生的认知能力，合作学习通过调动学生内在与外在的学习动机，促进学生的知识学习；二是学生合作技能的发展，合作学习本身包含了合作的技能，学生通过合作学习，可以建立合作意识，培养合作能力；三是发展学生的健全人格，合作学习能让学生向同伴充分地表达自己——作为一个完整的“人”，有利于学生的全面发展；四是发展学生的创造性，合作学习重视学生的差异，保护学生个性，在合作中唤起学生创造性。

  4. 对教师提出新的要求合作学习具有复杂性，课堂教学中各种动态因素让合作学习过程充满了未知，对教师把握课堂教学提出了很高的要求，教师在对课堂的“控制”与“自主”之间要掌握恰如其分的度：课堂教学既要保证合理有序也要气氛融洽，保证学生能够自由地展开合作。在师生互动方面，教师要通过学生的表达洞察学生的知识、技能、情感等三维目标的发展情况，并进行有针对性、高质量的反馈，否则，学生会对教师的反馈失去信心，导致合作兴趣骤减。合作学习也为教师指明了出路，师师合作，教师之间敞开课堂大门，通过课例研究，共同学习、进步。

  5. 发展完善理论体系相较于发展了数百年的传统课堂教学，兴起于20世纪70年代的合作学习仍然是一种新型的教学方式。在传统的课堂中应用合作学习，会有多方面的束缚。例如在教学方面，学生在合作学习过程中会得到多种多样的结果，教师能否成功地把这些不同的结果有机融入课堂教学，并对学生进行恰当的反馈；又比如在教室座位的安排方面，合作学习需要学生面对面的交流、讨论，或者需要一定的场地完成学生的合作，现行教室内的安排显然已经不能满足合作学习的需要。这些挑战都显示了合作学习发展的不成熟，仍然需要在教学实践的检验中，不断发展、调整、完善。作为一种新型的教学方式，合作学习意味着对传统教学形式的一次变革，在冲击传统教学模式的实践过程中，合作学习注定会遇到诸多保守因素的阻碍。正如迈克尔•富兰所说的那样，虽然“变革的过程非常复杂而且充斥着未知因素”，“但是我们可以学习和预先了解它和更有效地和它相处”。在新的起点重新审思我国合作学习理论的研究，为合作学习的再次发力埋下了伏笔。
