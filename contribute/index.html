Markdown 是个啥？

Markdown 是一门轻量级的标注语言，Markdown 的目标是实现「易读易写」。只要会电脑打字的，就能在五分钟内基本学会它的用法。现在有越来越多的撰稿人、作者开始使用这种语言进行写作。但请不要把这里提及的“语言”和你听说的各类编程语言做类比。比起复杂的C语言，Java等等，Markdown使用的符号几乎不超过十个。这门语言设计的初衷是让使用者专注于内容而不是格式。

Markdown 语法的简要规则

标题

大、小标题是每篇文章都需要添加的。在 Markdown 中，如果一段文字被定义为标题，只要在这段文字前加 # 号即可。

文本修饰

Markdown重点在于进行排版。文字的字体、字号等是无需设置的。但有三类格式是可以在Markdown内进行的。包括：

  • 用两个 * 包含一段文本就是粗体的语法

  • 用一个 * 包含一段文本就是斜体的语法

例如：这里是粗体 这里是斜体

引用

如果你需要引用一小段别处的句子，那么就要用引用的格式。

只需要在文本前加入 > 这种尖括号（大于号）即可

列表

在 Markdown 下，列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加 1. 2. 3. 符号要和文字之间加上一个字符的空格。

无序列表如下：

  • 第一行内容
  • 第二行内容
  • 第三行内容

有序列表如下：

  1. 第一行内容
  2. 第二行内容
  3. 第三行内容

图片与超链接

有时候，需要为一部分文本设置一段超链接，以便读者点击这部分文字就可以转换到其他页面。那么操作方法如下：将需要显示且添加超链接的文字用方括号括起来，如 [ 此处显示超链接 ] ，然后在方括后面用小括号括除需要跳转的网址链接，如 ：

[ 学习共同体联合会 ](http://slc.education)

图片无需在您的文稿中插入，请随附件单独打包发送到投稿邮箱。

哪些情况会被退稿

如果您收到我们的退稿或者拒绝邮件，可能是以下原因导致的。为了让您的文字能又好又快的发表，请确保第一次投稿前就避免了以下问题：

  • 只在word文档中插入图片，却没有以压缩包的形式另外提交图片给编辑部

  • MD格式文件出现大量错误带来庞大的编辑修改工作

  • 文章内容含有大量的抄袭

  • 文章内容与学习共同体联合会网站没有任何关联

  • 文章内容没有得到原作者或全部作者的授权
